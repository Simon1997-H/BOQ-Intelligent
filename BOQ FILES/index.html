Included Rules

Slab: area = length x width, volume = area x thickness, edge board = perimeter x thickness.

Footings: volume = plan area x height, side formwork = perimeter x height.

Walls: volume = length x thickness x height, formwork = two faces plus optional ends.

Radius elements: area = πr², volume = area x height, side formwork = circumference x height.
